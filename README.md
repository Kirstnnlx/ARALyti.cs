# ARALyti.cs
Study and Project Insight System

# Project Description
A Study and Project Insight System that analyzes your C# projects to track your progress and guide what to learn next.

# Purpose
A study and project insight system designed to help programmers monitor their C# programming progress. It analyzes uploaded C# projects to identify learned concepts, track topic development, and recommend what topics to study next for continuous improvement.

# Features and Functionalities
| Feature | Functionality |
|--------|------------|
| 📘 **Topic Tracker** | Track 12 structured topics (Easy → Hard) |
| 🔍 **File Scanner** | Detect concepts from uploaded `.cs` files |
| 🤖 **Recommendation** | Recommendations based on progress of the scanned topics |
| 📝 **Project Diary** | Store notes and code snippets |
| 📊 **Progress Analytics** | Visualize learning progress and statistics |
| 👤 **One-time Login** | Quick and simple access login |

# Explanation of how the program works

**1. User Login** - The user logs into the system using the one-time login feature to access the platform securely.

**2. Uploading a C# File** - The user uploads a .cs file or C# project into the system for analysis. 

**3. File Scanning and Analysis** -  The system scans the uploaded file and detects programming concepts, topics, and coding patterns used in the project. 

**4. Topic Tracking** - Detected concepts are matched with the system’s structured learning topics to monitor the user’s progress.

**5. Recommendation** - Based on the scanned topics and current progress, the system recommends what concepts or lessons the user should study next. 

**6. Progress Analytics** - The system displays visual insights such as completed topics, learning progress, and improvement statistics. 

**7. Project Diary** - Users can save notes, reflections, and code snippets related to their learning journey and projects.
